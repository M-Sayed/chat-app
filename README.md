# Chat Application
> A Chat application build with Rails and its ActionCable.

## prerequisites
* Ruby 2.4.0
* Rails 5.1.4
* Redis

## Setup & Usage
* clone the app.
* install gems, `bundle install`
* setup database, `rake db:create db:migrate db:seed`
* run server, `rails s -p5000`
* go to http://localhost:5000
